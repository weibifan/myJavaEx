### JDK
早期
Sun公司开源了Java的JDK，出现了OpenJDK7，sun公司基于OpenJDK7做了少量修改变成JDK7（或者JDK1.7）。后来sun被Oracle收购。
简单说，JDK1.8是Oracle维护的商业版本（含有不开源代码），OpenJDK8是社区版本（仍然是Oracle维护），各个厂商都会有自己的JDK版本。
jdk11少了jre的部分，而jdk1.8包含jre部分。是精简版与完整版的区别。
